Detailed Description

template<typename Request, typename Response>
class bond::ext::grpc::shared_unary_call< Request, Response >

A shared owner of the details of a single async, unary call.

Call Finish or FinishWithError to send a response back to the client.

To obtain a shared_unary_call instance, call unary_call::share or construct a shared_unary_call from a unary_call rvalue (e.g., via moving).

If no explicit call to Finish or FinishWithError has been made before all the shared owners are destroyed, a generic internal server error is sent.
